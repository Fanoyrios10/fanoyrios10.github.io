Καλώς ήρθατε/Welcome

Σε αυτόν τον ιστότοπο θα εξερευνήσουμε ιστορικά την ανάπτυξη των Windows ξεκινώντας από τα Windows DR5 (1984) και φτάνοντας μέχρι και τα Windows 8.1 (2014). Επίσης θα κοιτάξουμε και αρκετές δοκιμαστικές εκδόσεις των Windows για να δούμε πιες αλλαγές έγιναν πριν την τελική έκδοση των windows. Θα δούμε μέσα από αυτές τις εκδόσεις, τις προσθήκες λειτουργιών, τις διαφορές και τις βελτιώσεις των προγραμμάτων που έρχονταν μαζί με τα windows χρησιμοποιώντας κείμενο, εικόνες και βίντεο. Αν και θα ήθελα να χρησιμοποιήσω υπολογιστές εκείνης της εποχής διστυχώς δεν έχω πρόσβαση σε τέτοιους υπολογιστές και για αυτόν τον λόγο θα χρησιμοποιήσω προσομοίωση κυρίως μέσω του προγράμματος Pcem. Συνολικά θα κοιτάξουμε περίπου 40 διαφορετικές εκδόσεις των Windows. Σε όλες τις εκδόσεις των Windows θα σας δώσω αναλυτικές οδηγίες για το πως γίνεται η δημιουργία ενός εικονικού συστήματος για την συγκεκριμένη έκδοση των Windows καθώς και οδηγίες εγκατάστασης σε μορφή κειμένου, εικόνας και βίντεο εάν θέλετε να εξερευνήσετε τις εκδόσεις των Windows.

In this webpage we will be exploring the development history of Windows beginning with Windows DR5 (1984) and ending with Windows 8.1 (2014). We will also be looking through several of the preview and beta builds of Windows so we could see the difference between them and the shipped product. Through these versions of Windows, we will be seeing what was added, what was cut, what was improved etc. We will be seeing the main programs that shipped with Windows and how they evolved over time. Apart from text I will be using plenty of pictures sound files and video. Although I would love to have taken this trip with history by using hardware from all the eras I will be covering I don’t have that hardware so I will be using emulation, mainly PCem. In general we will be looking at around 40 differnt versions of Windows. In all the versions of Windows I will be giving you detailed instructions of how to create an emulation system and how to install Windows in case you want to explore them for yourself.
